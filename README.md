Este projeto foi elaborado com a utilização da persistência de dados e a utilização da API de armazenamento do navegador, através do Local Storage. Utilizei métodos para manipulações dos dados de notas de alunos fornecidos pelo usuário através do preenchimento de formulários. O fluxo de trabalho segue com a inserção dos dados pelo usuário, pegando esses dados armazenados no Browser.
As ferramentas e tecnologias utilizadas para a construção dessa aplicação foram HTML, CSS, Javascript Vanilla e o Google DevTools.
